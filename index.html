2016年终总结

发表于 2016-12-31 | 分类于 blog

虽然已是冬天，但天气却着实暖和，早晚的寒风，偶尔透露着年终的将至，校门口光秃秃的落叶树，就像时间一样，说落就落了。

2016，家庭琐事依旧挺多，也挺让人心烦，但生活本就如此，自然也就没啥好总结的了。感情方面，还没遇见那个有缘人，我想她也正在为我努力着吧，正如我正在为她努力着。但今年是我参加工作的第二个年头，作为老教师，面对着如此的成绩，让人十分不是滋味。

上半年，我由教三个物理班级改为了两个，的的确确，感觉更好上课，更好改作业了，精力也更加集中，但成绩依然不如人意。可能学生已经习惯我对他们的宽松管理，以至于课堂效果不佳，作业完成质量不高。

下半年，已经是我当教师的第二年了，新带一批学生，不同的面孔，我也有着不同的心态。初一生物，我的专业，我尽力带着一种培养兴趣的心态教他们，尽力让他们体验到更多。希望他们能更多的学习到有用的东西，了解生活中的相关生物知识，保持对这个世界的好奇。学生们是开心了，可我依然闷闷不乐，对成绩不是很满意。初二物理，第二次教，已经对课本比较熟悉，了解了重难点，知道哪些是常考点，可依然教不好这一科。

今年，学校破天荒的让各班举办了元旦晚会，大家也都过的比较开心。也让我看到了学生不同的一面，不同于平时的拘谨，有些还才艺尽显。我也收到了一些明信片，让我更加的了解了一些学生，以及他们对我的看法。有些是上一届的学生，看样子以前的教学也不是一无是处，这让我很是欣慰。

成绩出不来，一方面是我的问题，另一方面是学生。学生最大的问题是没有学习目的，没有学习目标，没有学习动力。一部分学生做一天和尚撞一天钟，一部分直接自暴自弃，剩下三三两两自觉性比较强的在学习。有时我不得不感叹，祖国的花朵并不是都可爱，有一些甚至已经枯萎了，他们并配不上老师的尽职尽责。

新的一年，希望学生都能有所成长，学业有所进步。

期中总结

发表于 2016-11-27 | 分类于 blog

时间如流水，不舍昼夜，眨眼间，期中考试已然结束，成绩有点惨不忍睹。新的一批学生，本以为，有了一年经验的我，可以有一个稍微好一点的成绩。但令我意外的是，竟然比上一届学生还要考的差。更可气的是，两个班相差了8分，差的一班与全校第一名班级相差13分，及格人数方面，一个班20人，一个班8人，差距之大，令我费解。曾经的我，还可以用没经验当借口，可以自认为是对课本、对考点、重点不熟的原因。一年后的今天，有这种结果真的很是不能理解。思前想后，我也只能从自身找原因，继续改进教学方法。

对于基础题型，后进生格外令人失望，原题、同类型题目都无法有个高的正确率。经统计，整张试卷有60多分是基础题型和原题，但平均分却远低于此。今后上课要多注意学生的上课状态，多关注学生。严格要求学生，对于原题，应保证不失分。

新课程，新要求，部分学生对答题规范不了解，了解了也不重视，导致计算题失分严重。后期定要强调答题规范性，多重复内容，反复练习。

实验探究题，是学生的难点，操作做的少，对过程不了解，无法答到要点。课堂演示实验应尽量详细，多多提问，加深学生的印象。平时作业加强这方面的练习，及时进行讲解，巩固。

优秀的学生有点偏少，需格外分层出对应的习题，培养优生，及时进行辅导。对于低分段的学生，多出基础题，保证拿到可拿的分，稳步提高成绩。

纪律方面还需加强，对于个别调皮捣蛋的学生，严肃处理，正确惩戒。希望下半个学期能有一个较好的进步。

教育的目的

发表于 2016-11-14 | 分类于 blog

我们校长说，他认为教育的目的有两个：

使人有谋生的手段

使人有一颗感恩的心

的的确确，人归根结底，最基本的需求是活着，生容易，而生活却并不简单。要生活的好，生活的有质量，那就需要某种技术，某种谋生的手段。而教育，可以快速的让学生学习基础知识，广泛的了解先人留下的知识经验。更重要的是，可以学习到学习的方法、技巧。毕竟，如今的世界，去哪都是需要学习的，学习的慢，跟不上脚步，终将会被淘汰。

而人如果没有感恩的心，不知感恩父母、老师、良友，那是相当可怕的。人之初，性本善，而教育的目的，也是为了维持这种善，使人不至于变的内心丑陋。我一直相信学生是可爱的，也一直希望学生走出了校门，仍然能够可爱，怀揣一颗感恩的心去拥抱这个会越来越美好的世界。

《致教师》读书心得

发表于 2016-10-25 | 分类于 blog

《致教师》全书共分四辑，分别是“给我一个做教师的理由”、“借我一双好老师的慧眼”、“愿我书写一部教师的生命传奇”、“让我过一种幸福完整的教育生活”。前三辑用书信问答的方式呈现，解决了很多老师在工作中遇到的难题，遇到的困惑，每个问题都详尽细致，深入浅出的进行了剖析。对我这个新老师来说，读罢此书，受益良多，也更加坚定了我从事教育的信心。

书中写到，教师职业大致有四种境界，一是让学生瞧得起的老师，二是让自己心安的老师，三是让学校骄傲的老师，四是让历史铭记的老师。而我正处于第一种境界，如何让学生瞧得起。“学高为师，身正为范”，一方面，我得努力提高自己的教学能力，让学生更容易的理解所学知识，提高学生对课程的兴趣、积极性。也只有学生的成绩上去了，学生才会佩服教师的教学能力。这就需要我用心去写每一个教案，用心去教每一门课。另一方面，对自己的要求要提高。教师的言行是学生活生生的教科书，学生很容易模仿老师，以老师为“榜样”，因此，要做到为人师表，每时每刻都要注意自己的言行举止，课前课后都要给学生树立一个好榜样。

另一段“你的胸怀决定着教室的大小”也令我印象深刻，教育不光是给孩子们知识，更重要的是培养学生一种积极的生活状态，以积极的生存心境、积极的人生态度对待生活。然而现今的我们却不得不一味的关心名次、分数，而去忽略更大的世界。可能素质教育在我们这个人口大国暂时还比较遥远，但也希望可以有更多的学生去关心教科书以外的东西，关注这个正在剧烈变化的社会，关注国家民族的大事情。

当你接触过教育，特别是山区的教育，就注定着一辈子无法忘记教育，无法忘记山里的孩子，希望自己在教育的路上越走越远。
