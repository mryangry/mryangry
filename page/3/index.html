有人问我

发表于 2016-01-05 | 分类于 blog

　　有人问我，如果看不到确定的未来，还要不要付出。

　　我只能说，并不是每一种付出都是在追寻结果。有时在付出的路上，能够收获的，是清楚的看到了自己想要的，或者不想要的，这又何尝不是一种宝贵的结果。命运会厚待温柔多情的人，好过冷漠的一颗心。

　　有人问我，是不是应该为了一个人，去一个陌生的城市。

　　我只能说，有时候你不是爱上了一个人，你只是爱上了一种生活，你希望幻想中的生活里，有这样一个人。所以不管留在哪个城市，这种生活总会存在心里，而那个人也许换了不同姓名。与其说你为了一个人，不如说是为了自我的一种私心。牺牲总是听起来伟大，但谁又能无所求呢。若是看到了自己所求的，也就能平静的离开，饮鸩止渴也甘之如饴。

　　有人问我，是不是该等待。

　　我只能说，等待是爱情里最美的一种姿态。但这种美好是等待的每一天里，你都变得越来越好，越来越温暖。如果只是静止成为一种单调而重复的生活，那就辜负了等待的意义，而变成了另一种私心，一种等待被救赎的懒惰，懒惰的人从来不会有幻想中的丰富而安然的生活。

　　有人问我，孤独的时刻应该怎么度过。

　　我只能说，如果没有人可以帮你，那么你能做的，就是尽力去帮助其它的人。这世界奇妙的很，我们总是收到偶然中来自他人的帮助，而给予者往往比接受者更强大。变成一个给予多过接受的人，当你变成某种意义的强大，孤独就不再是一种煎熬。

　　有人问我，为什么无法控制自己的感情。

　　我只能说，我们生来不是为了要满足某个人，不是为了某个甲乙丙丁的心情而存在。我们来到这个世上一定有更深的意义，就算最终是碌碌无为的一生，也一定有一个闪光的时刻，看到隐藏在其中的一些启示。也许某些人的出现会短暂的控制了你的心，不过在漫长的人生里，这些也只是短暂的一个片段。

　　不管你问我什么，不管我说了什么，除了能让你略感平静，其实没有别的意义。

　　一个人是什么样子，他或她很久以前就是那个样子，只是你没有发现而已，没有人会一夜改变，所有结果是一个又一个自我选择的叠加。

　　而话语是最卑微的事，改变不了什么。

2015年终总结

发表于 2015-12-31 | 分类于 blog

今年，年初，我决定好好的考教编，便先去了姐夫厂里，准备边干点活，边复习下。干体力活还是蛮累的，但我还是吃的消，大学期间的每年暑假我也是在那做事的。身体的累是可以承受的，但没过几天，也就是在元宵节时，发生了一件事，我很伤心。这是我始料未及的，没有一点点征兆，可能是前期的种种酿造的后果吧，这让我度过了一段伤心难受的日子。在这种刺激下，以及我的努力下，我还是没考上萍乡四中。迫不得已，经过我的数据分析，我调剂到了人少分底的革命老区瑞金。不出所料，我幸运的过了，面试那天还发生了一段小插曲，这也就无关紧要了，就这样我在瑞金当上了一名人民教师。

来学校的路上，我是烦心的，从没坐过时间如此之长的巴士，山非常的多，路非常的烂。刚到学校，我是震惊的，从没有见过这样的初中，杂草非常的多，房子非常的烂。但我还可以接受，不像其他人说的那么夸张，会流泪。这也许是我未来工作一段时间的地方，我会在这由黑板、讲台、粉笔和书本组成的世界里，和学生们一起前行。既然来了，那就好好的做好。

刚做老师，经验非常的不足，结果，成绩非常的差。我也在教学中，慢慢的提升着，和这里的学生接触的越久，也就对他们越关心，留守儿童居多，家长把孩子放在这，那也是一种信任。整个下半年，听了非常的其他老师的课。也许成绩不好跟我的管理有很大关系，也可能是不够努力吧，希望明年可以有个比较大的进步。

期中考试总结

发表于 2015-11-25 | 分类于 blog

　　期中考试已过，教学还得继续，对于第一次交成绩的我，结果固然是让人失望的。及格率太低，平均分也低，种种的不理想都集中体现了。我想当下分析原因，吸取教训应该是我必须得做的事情了。本着没有蠢学生，只有蠢老师的原则，对这次期中考试总结如下。

　　作为新老师，我缺乏经验，一些教学方法依然无法熟练的运用。对于非本专业的物理，前期了解的不够，对课本内容不熟，重点知识的梳理也不是很好，有些问题想的过于复杂，学生不易理解。有些问题又讲的过于简单，使学生理解的不深。把教学这个工作想的太简单，学生想的太聪明，对学生学习能力参差不齐的现状，没有去理解，应多站在学生的角度去思考问题，照顾到更多的学生。

　　前期讲课过快，内容讲解的不深，使学生的基础不牢。虽然学生做的习题非常的多，但质却没有提升，一些后进生无法按质完成作业。课堂不够精彩，实验多以演示实验为主，学生的实验探究比较缺乏，不容易让学生接受，与学生的互动也不多，需要加强。

　　纪律方面，要求的不严格，而学生的思想意识差，自觉性不高，导致上课纪律比较差，影响了教学效果。对未按时按质按量完成作业的学生，处罚力度不够学生主动学习的热情不高。在下半个学期，纪律必须管理好，保证学生有一个良好的学习环境，才能有助于他们的学习。

　　对培优抓及格方面，没有掌握合理的方法，虽然出了分层次的试卷，但学生的完成度不高，没有达到应有的效果。自己的做题量不够丰富，常考题型没有把握好。期中考试复习过程中，虽然打印了复习重点，却没有进行详细的讲解。

　　在后续的教学过程中，我将改进教学方法，多精选习题，定期对教学效果进行检测，以期学生能有一个较大的进步。
