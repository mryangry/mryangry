代理班主任结束了

发表于 2015-11-09 | 分类于 blog

　　同事老婆生孩子，请了一个礼拜的假，而我是那个班唯一没当班主任的男老师，代理班主任的任务就自然而然的落到了我的头上，没当班主任的我提前体验了一把班主任的日常。

　　我最无法适应的可能是早起了，平时7点准时起床的我，得提前一个小时，天都还是蒙蒙亮的，凉意十足。学生们站完队，听完值周老师的讲话，还得晨跑一段距离，而我为了不让宿舍扣分，会在这个时间去查下宿舍。之后便是去教师看一看，早晨班主任的必要性体现的不是很明显。晚上睡觉时得去查寝，督促学生早点休息，不卧谈。

　　这个班是学校出了名的调皮，我也做好了心里准备，没想到的是前两天就扣了5.5分，一时手足无措的我，想尽一切办法，终于稳定了下来，在倒数第二天扣了0.5分，结果我还是挺满意的。中间也发生了非常多的事情，最让我头疼的是学生上课不听话得罪科任老师，结果处理的不理想。期间还有个学生胃疼，联系家人费了好久的时间，最后还是通过她家邻居联系上的。期间有几个学生不听我的话，最后我把家长叫过来了，这个可能是我最没处理好的。学生不听话必定有原因，叫家长应该是下下策了，事后心里总有点过意不去，还是经验不足。

　　总的来说，当班主任是非常累的，班里大大小小的事都要管，碰到个急事，吃不上饭，是最正常不过了。但也是最有收获的，看着自己带的学生越来越好，一个个洋溢着笑容的青春面孔出现在你面前时，也会欣慰的露出笑容吧。

包饺子

发表于 2015-10-25 | 分类于 blog

今天上午留校的老师一起包饺子了，香菇肉馅的，放了非常多的葱，这应该是我第二次包饺子了。第一次还是大一时吧，那时是全班同学，包，非常的热闹。这次虽然人少，但分工明确，每个人都乐此不疲的，配的汤料比较简单，但相对每天菜式没咋变化的校园食堂，这顿饺子真的是一顿大餐。:)

做米糕

发表于 2015-10-24 | 分类于 blog

又是一个周六，这次留校的老师是最多的一次，有10多个，闲来无事便做起米糕来。米糕，下坝的特色食品，第一天来的晚上，我就尝过了。上次用的是豆子叶，这次是红薯叶。将米粉磨成米浆，和着红薯叶成块的蒸，非常的清香。老板娘配的辣椒料水，全是本地常见的朝天椒，非常的辣。用米糕蘸着吃，是本地人无法拒绝的，非常有特色的一种食品。

爬山

发表于 2015-10-17 | 分类于 blog

今天下午，我们去爬山了，山虽然不高，但路非常的崎岖，非常的远。比国庆时爬的矮多了，但对我来说却累了好多。爬到山顶，看到了整个区域的部分面貌。一眼望去，是连绵不绝的山，起起伏伏的，有一种壮丽山河的感觉。令我感到吃惊的是，半山腰处依然住着10几户人家。接近山顶处，有一个天然气管道隧道，属于西气东输工程的一部分，应该有3000米吧。我们走了好久穿过了它，我期待着可以看到一个世外桃源般的小山村。对面的确有个小村子，稀疏的坐落着十几栋房子，有土房，也有砖房，风景还不错。

白水寨瀑布

发表于 2015-10-11 | 分类于 blog

因为国庆放假调休的缘故，这周周六补课了，只有今天一天休息，所以大部分老师都留校了。我们上午去了附近的白水寨，那里有一个没有开发的大概30米高的瀑布，据说这里还有一段凄美的爱情故事。中午时我们去了曲阳吃饭，这是一个被老师们多次提及的地方，也是同事们经常去的地方。久待学校的我们去那吃顿“大餐”，真的是一种享受。吃完还在附近河里旁的沙滩玩了一会，赤脚踩在沙滩上，会给人一种在沙漠的错觉。一整天，大部分时间都用在路途中了，但依旧非常的开心。

报到

发表于 2015-08-31 | 分类于 blog

　　今天上午在教育局选的岗，刚选完，便找到了接头的老师，下午就得直接去学校。

　　下午我们如约来到了约定的汽车站，还有另外两位新老师，一男一女，带我们去学校的是一位美女老师。我和男老师一起到的，上车时，她们帮我们占好了位子。车上非常的挤，坐满了人，行李也异常的多，大概这已经预示了路途的遥远吧。结果果然如此，非常的远，路还非常的烂，14:00出发的车，17:00点样子才到。真是一路十八弯，一摇一摇的。

　　到时司机师傅还绕路把我们送到了学校，学校是“藏在”一片竹林里的，黄色的房顶若隐若现。见到的第一眼，我心里真的有点失望，这竟然是个初级中学，操场上长满了杂草，花坛里也是十分的不整齐，一栋3层的房子应该是教学楼，旁边还有一栋瓦房，很旧的那种，乒乓球台倒是很多。这便是我这段时间工作的地方，一年或者几年。

　　傍晚的时候，学校老师在附近一老板的邀请下，去喝了“擂茶”。这是由柚子叶，花生，芝麻等榨的汁，有些喝不习惯的会喝不下，柚子叶的清香非常的浓。但对于我这种啥都吃的来说，当然不在话下，就和和一杯杂粮豆浆差不多。作为新老师，当然会被问及一些感受。也会听到“老”老师的经验之谈，这里的老师大部分都是像我一般年龄的年轻老师。通过他们对这学校也便有了个初步的认识，希望可以在这里有段愉快的日子。

苏珊与薛海

发表于 2015-07-24 | 分类于 blog

　　昨晚上，爷爷给我讲了一件事儿，说他年轻的时候，邻居家的女儿很漂亮，名字叫苏珊。此人的家境，其实从名字便能看出几分，一定是受了西洋文化的熏陶，才起了这么个洋文名字。

　　爷爷一直都挺欣赏苏珊的，唯独看不惯她的大小姐脾气。那时候爷爷写得一手好诗，苏珊也很欣赏爷爷，加上她又会拉小提琴，所以他俩就联合办了个配乐朗诵沙龙。

　　爷爷从小就经常在另一条街玩，那条街是我们家里有名的“酒吧街”，我爷爷特别爱去那条街和一堆小伙子喝酒。听说爷爷要办这个沙龙，那帮“狐朋狗友”自然想一睹苏珊的风采，就都来参加了。

　　这帮朋友当中有个爷爷的铁哥们，叫薛海，薛海是个农家小伙子，却不太普通。之所以说他不太普通，是因为别的农家小伙子都放牛养狗，而他却养了一头鹿。那鹿是他有次去野外玩耍时捡来的。后来就一直养大了。他给那鹿起了个名字叫狗狗。

　　可谁知这沙龙举办第一次，苏珊和薛海一见钟情，好上了。他们俩非常恩爱，叫所有人都羡慕不已。可好景不长，由于门不当户不对，苏珊的父母极力反对二人的交往，把苏珊天天关在家里。但这也没用，苏珊总能想到办法偷偷溜出去，患难见真情，俩人的感情反而比以前更好了。只是见面频率肯定不如从前了。大概一个月见一次，和现在的北京天津异地恋，北京通州石景山同城恋的见面频率也差不多了吧。见不到面的日子里，薛海和苏珊给对方写信，薛海每次写好信，就把信绑到狗狗的鹿角上，狗狗便跑到苏珊的窗口，苏珊从窗口探出手来取了信，又把自己早已写好的信绑在鹿角上。狗狗就当起了二人的信使。

　　这样的日子过了一段时间，见面机会的减少并没有影响到二人的感情，苏珊的父母没辙了，也就睁一只眼闭一只眼。可有一天早晨，苏珊终于又找到了借口溜了出去，迫不及待地来到她和薛海约定见面的老地方，却等来了一个令她终身难忘的噩耗。

　　那天，苏珊等了薛海很久，薛海也没来，小鹿来了，鹿角上照样有封信。苏珊问小鹿，狗狗，海大哥呢？他怎么没来。小鹿当然没法回答她。苏珊取了信，展开一看。

　　原来薛海的父亲为了躲债连夜搬走了，薛海也不知道父亲要带自己到哪里去。但是薛海说了，他一定会回来找苏珊的。他把小鹿留作礼物送给了苏珊，希望苏珊能够好好照顾小鹿。苏珊绝望地瘫坐在地上，有气无力地带着鹿回到了家里。

　　后来，过了很多年，薛海也没有回来。苏珊就一直坐在屋子里，整日流泪度日，把父母都急死了。苏珊后来渐渐觉得好了，就帮着打理家业，决心不考虑感情问题，做个女强人。有一天她在管家屋里整理旧物，忽然之间发现了一张欠条。原来，当年逼走薛海的债主，就是自己的父母。苏珊顺藤摸瓜调查，才知道当年父母使用了卑劣的手段，放高利贷逼走了薛海一家。苏珊感到羞愤极了，觉得自己竟然蒙在鼓里这么多年，于是父母安排的亲事，她统统都以死相拒。

　　过了好多年，苏珊一直没有结婚，每天都悉心照顾着小鹿，等着薛海回来找她。

　　“那这个薛海现在怎么样了！”我问爷爷。

　　“薛海在外面结了婚，听说苏珊一直等着他，不敢回来啊。我去看过他。他现在都老了，牙也掉光了，成天煮粥喝。总是一副苦笑的样子。”

　　我听完之后很感慨，若有所思地说：“苏珊有鹿情未尽，薛海无牙苦做粥。”
