老姐给我的加油信

发表于 2006-05-20 | 分类于 怀念

　　姐自高三以来，很少和你聚在一起。你我都将面临人生的选择。我这次回家，也不能和你好好谈谈，只好写好，望你细看。

　　我有预感，只要你能考上XX中学，考名牌大学对你来说不在话下。因此你一定要考上上栗中学。在如今这样的关键时刻，心理战是很重要的。切忌紧张灰心。要时刻勉励自己，要对自己说：“我都考不上，谁还能考上？”一定要有志在必得的勇气和信心。当然你也不能有丝毫的放松，要一如既往的学习，不要小看他人和中考。

　　姐虽不敏，亦得了一些学习之法，望弟赏鉴，并有所获得。

　　如今离中考只剩20多天了，你没有必要再去钻难题，新题。要看以前所做的题，稍有不懂，一定要问别人。还有就是一定要重基础，要勤翻课本，因为所有的考题都来源于课本，课本都不熟悉，你还做什么题。因此，你一定要注重基础。

　　下面我就谈谈语文和英语的学习方法吧

　　语文是一个注重积累识记的学科，首先，你一定要掌握基础知识。从现在起，你每天都要看一份以前做过的语文卷子，把前面几个小题中考查了的字词音义记清、记牢，每天早上还要读一读一些你觉得考试会考这一类的阅读理解，多读读作文，看看别人是怎样写开头和结尾的，更要记清记准一些事例和名句。不知你们有没有默写题，若有，你不能只背，还要默写出来，看哪些字写不出。如果你能坚持每天这样做的话，你的语文成绩会更上一层楼。

　　英语和语文有些相似。你一定要多花点时间，每天早上要多读做过的题，完形填空、阅读理解、书面表达都要读，单词也要记。这样才能培养语感（即看到这个题目就能选出正确答案，虽然不知道为什么要选这个，但只要选对了就可以）。听力作为第一题，是丝毫不能小看的，如果听力没做好，肯定会影响情绪。你要有这样一种思维，即发下试卷来，你一看到听力题目，就能想出他会给出什么材料。单个字词不清楚，不要灰心，这是正常的，一定要轻松认真的从头听到尾。早晨也可以拿出英语报纸，对着题目，读材料。相信你的英语也会更上一层楼。

　　适当的休息是必要的，你没有必要每一分钟都学习，但一定要没学习一分钟都收到一分钟的效果，不要在乎别人的看法，要自信。

　　姐相信凭你的功底和信心，一定能考上江西省重点中学——XX中学。

　　姐也一定要发挥自己的水平，考上理想的本科大学。

　　让我们为了心中的梦想，拼上一回吧！

# 中考 # 信件
